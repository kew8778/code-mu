Задача №3

            <input type="button" id="elem" value="1">

            let elem = document.querySelector('#elem');

            elem.addEventListener('click', function() {
                setInterval((function(self) {
                    return function() {
                        console.log(self.value);
                    }
                })(this), 1000);
            });

По клику на кнопку вызывается анонимная функция, внутри которой вызывается таймер setInterval.
Таймер выполняет каждую секунду функцию function(self), вызываемую на месте с переданным в параметр self объект this.
Сама функция function(self) возвращает функцию, выводящая в консоль значение value объекта this
